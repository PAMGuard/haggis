Haggis Detector

Overview

Help files for PAMGuard are in html format. You are advised to look at examples for other detectors, which are all in sub folders of 'help' before writing your own.

For plugin modules, you'll need to create a Java helpset. This is based around the JavaHelp system for which there is plenty of documentation online.

The PAMGaurd team have been using jhelpdev which seems to work well enough, though it may be a but out of date. jhelpdev will help you to automatically create the requires .hs helpset file and required xml files for content and indexes. The files for this are all included in the src/help folder. If using jhelpdev, open the file DemoDetectorHelpProject.xml. Note that you will probably have to change some of the paths in this file before you can use it.
